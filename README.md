# Документация по Jarvis

## Установка
Чтобы установить Jarvis, выполните следующие шаги:

1. Убедитесь, что у вас установлены необходимые зависимости:
   - Python 3.x
   - pip

2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Xatab701/Jarvis.git
   cd Jarvis
   ```

3. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

Для того чтобы запустить Jarvis, выполните следующую команду:

```bash
python main.py
```

### Пример команд
Вы можете использовать следующие команды:
- `привет` - Получить приветствие
- `помощь` - Показать доступные команды

## Команды

Вот список доступных команд и их описание:

| Команда     | Описание                          |
|-------------|-----------------------------------|
| `привет`    | Получить приветствие             |
| `помощь`    | Показать список команд           |
| `выход`     | Выход из программы               |

## Устранение неисправностей

### Проблема 1: Jarvis не запускается
- Убедитесь, что все зависимости установлены. Попробуйте переустановить зависимости с помощью команд `pip install -r requirements.txt`.

### Проблема 2: Неизвестная команда
- Пожалуйста, проверьте правильность ввода команды. Чтобы получить список доступных команд, введите `помощь`.
